## Project Setup

1. Install dependencies:

    ```bash
    yarn install
    ```

2. Copy environment variables:

   **For Linux / Mac OS:**
    ```bash
    cp .env.example .env
    ```

   **For Windows OS:**
    Manually copy and paste the `.env.example` file into a new `.env` file.

3. Run the website:

    ```bash
    yarn dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the website.
